# TTS Supported Languages and Voices

This document lists the languages and their corresponding voices supported by the Text-to-Speech (TTS) feature of the application.

The application supports two TTS engines:
1. **Kokoro TTS** (Local, Free) - Runs on your computer via Docker
2. **Google Cloud Text-to-Speech** (Cloud, Requires API Key) - Optional, requires Google Cloud account

## Notes

*   **"Text Only — no TTS"** is also an option, which disables audio output.
*   **Kokoro voices** run locally and are completely free with no API costs.
*   **Google Cloud voices** require an API key and may incur costs based on usage (see Google Cloud pricing).
*   Voice names often indicate gender and region (e.g., `af_heart` = American Female Heart, `en-US-Neural2-A` = English US Neural voice).
*   Google Cloud offers three quality tiers: Standard (basic), Wavenet (high quality), and Neural2 (most natural).

This document was generated based on the `main.py`, `translator_core.py`, and Google Cloud Text-to-Speech API documentation.
